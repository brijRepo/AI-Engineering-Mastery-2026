#Variables
Stores data values.

Example:
data = "info"

AI Usage:
Stores matrices or tensors,
Context, Session states, Chat history,
Configuration,
Temporary streaming data, etc.



#Lists
Ordered sequence of items in []

Example:
list = ["name", "id"]

AI Usage:
Data stream collection, Sequential text processing,
Batch management, etc.



#Tuples
Immutable, ordered sequence of items enclosed in ()

Example:
image_dimension = (64, 3, 244, 244)

AI Usage:
Define Model Architecture, Track mathematical grid dimension, etc.


#Sets
Unordered collection of unique items enclosed in {}
Lookup is O(1)

Example:
blocked_toxic_words = {"harmful_word_1", "harmful_word_2"}

AI Usage:
Enforce uniqueness, De-duplicate massive datasets, etc.


#Dictionaries
Stores data as key-vale pairs

Example:
user = {"name": "Brijesh"}

AI Usage:
API Responses, JSON payloads, Configuration files, etc.
